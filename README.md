# Dispromax - Gestión de Pedidos

Aplicación web profesional para gestión de pedidos de una distribuidora.

## 🚀 Tecnologías utilizadas

- **React** + **Vite**
- **Tailwind CSS**
- **Supabase** (Base de datos y backend)
- **jsPDF** (Generación de PDFs)
- Totalmente compatible con **Vercel**
